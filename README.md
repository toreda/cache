# `@toreda/cache`
Simple TTL-based object cache in TypeScript.

&nbsp;


# Install
`@toreda/cache` is available as an [NPM package](https://www.npmjs.com/package/@toreda/cache).

&nbsp;

**Install with yarn:**
```bash
yarn add @toreda/cache
```

**or Install with NPM:**
```bash
npm install @toreda/cache
```
